文章目的

  1. 开发的学习道路（计算机基础环境原因 语言 需求 工具 进阶）

    • 原理：页面渲染（浏览器的工作原理）+ 数据通信（请求响应）+后端的业务处理+数据库操作 ==> 页面请求+后端服务+操作数据库+后端服务+后端响应+前端整理数据+页面渲染
    • 工具：（超文本标记语言html+css样式+javascript+service+jsp+javaweb+DB）
    • 开发进阶（：前后端分离 +Nginx端口连接+JSON(原名：JavaScript Object Notation)+axios(基于 promise 的 HTTP 库，简单的讲就是可以发送get、post请求)+SSM+

  2. 快速上手开发（举例子基础的 请求 服务 数据）+发展的动机（需求增加、流畅访问、稳定安全）

    • 登录界面+记住我（需求）+单点登录+（加密。第三方。令牌。
    • 解析cookie、session、解析token、filter过滤器

  3. 领略一些开发概念（前端环境框架、前后对接（端口和路径的问题）、数据库的设计（软件工程）、一些成熟的系统设计例子（安全加密/认证授权））

    • 开源网站上找已有的模板开发， admin.vue 这个板子不错，写后台控制页面用的（了解目录结构
    • 按照软件工程的开发流程：需求分析-数据库设计-概要设计-具体实现-测试-运维
    • RSA加密、sha256、md5+盐、Oauth2第三方api权限框架、springSecurity授权框架、shiro等等

基础开发

前端+后端+缓存/数据库

Html Css JavaScript Jsp Javaweb MySQL

进阶开发

RestFul风格、软件工程、单元测试()

后端框架（Spring、SpringMvc、mybatis）、前端（vue.js、node.js、element-ui）、第三方API服务（阿里云OSS、微信登录、支付、验证码）

高级开发

系统设计（认证授权、定时任务、实时消息）

分布式（注册中心、配置中心、RPC/feign、gateway、分布式ID&锁）

高性能（CDN、负载均衡、数据库优化、消息队列）

高可用（限流、降级熔断、超时重试、性能测试）

未来趋势

k8s+docker

服务环境

Linux、逆向工程、自动化运维、Docker

开发规范

《阿里巴巴开发手册》、分包命名、统一结果/异常/工具类、状态码、围绕文档

工具

Maven、git、GitHub、ChatGPT
